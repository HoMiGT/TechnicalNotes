# 1. 常见的构造函数
| 函数                   | 功能                                 |
|----------------------|------------------------------------|
| Tensor(sizes)        | 基础构造函数                             |
| tensor(data)         | 类似于np.array                        |
| ones(sizes)          | 全1                                 |
| zeros(sizes)         | 全0                                 |
| eye(sizes)           | 对角为1,其余为0                          |
| arange(s,e,step)     | 从s到e,步长为step                       |
| linspace(s,e,steps)  | 从s到e,均匀分成step份                     |
| rand/randn(mean,std) | rand是[0,1)均匀分布,randn是服从N(0,1)的正态分布 |
| normal(mean,std)     | 正态分布(均值为mean,标准差是std)              |
| randperm(m)          | 随机排列                               |
